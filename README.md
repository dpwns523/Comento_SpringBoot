# Comento_SpringBoot 

코멘토 직무부트캠프를 통해 Spring과 Spring Boot 환경을 직접 세팅해보고 기술 가이드, API 문서 작성

JAVA코드와 SQL을 혼용하여 필요한 데이터를 뽑아내는 실습을 한 프로젝트입니다.

개발환경 셋팅

    1-1. JDK 1.8

    1-2. springframework.boot 2.7.1

    1-3. mybatis.spring.boot 2.2.2
    
    1-4. Maven 
    
기술 세부 스택

Spring Boot

    1-1. Spring Web
    
    1-2. Spring Boot DevTools
    
    1-3. Lombok
    
    1-4. jstl
    
    1-5. maridb
